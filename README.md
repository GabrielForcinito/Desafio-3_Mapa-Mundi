# 🌍 Desafio Mapa Mundi

## 📌 Sobre o projeto

Projeto desenvolvido em HTML com o objetivo de praticar a organização de imagens e montagem de um layout simples representando um mapa-múndi.

## 🛠️ Tecnologias utilizadas

* HTML5

## 🎯 Funcionalidades

* Exibição de imagens em sequência
* Organização visual em formato de grade
* Estruturação de layout com boas práticas

* Inserção de múltiplas imagens
